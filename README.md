# FinSightAI

FinSightAI is a modern, AI-powered personal finance and portfolio analysis platform designed to help users track investments, gain insights, and make informed financial decisions through clean visualizations and intelligent summaries.

---

## Project Info

**Live URL**: https://<your-vercel-deployment-url>

---

## Features

- Secure authentication and user-specific dashboards
- Portfolio tracking and allocation insights
- AI-powered summaries for portfolio performance and market trends
- Weekly rebalance checks and recommendations
- Financial news analysis and summarization
- Responsive, modern UI with a clean design system

---

## Tech Stack

This project is built using:

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Backend**: Supabase (PostgreSQL, Auth, Edge Functions)
- **Deployment**: Vercel

---

## Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js (v18 or later recommended)
- npm

You can install Node.js using nvm:  
https://github.com/nvm-sh/nvm#installing-and-updating

---

### Running the Project Locally

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd FinSightAI

# Install dependencies
npm install

# Start the development server
npm run dev
```
